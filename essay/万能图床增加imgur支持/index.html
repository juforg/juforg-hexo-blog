万能图床增加对国外的图床imgur支持，使用说明

What?

imgur 是个什么鬼

两点信息 :

  1. <u>专门的图床网站，跟阿里云腾讯云的对象存储比 职能更单一</u>
  2. <u>国外的，而且 被墙了</u>

Where?

https://imgur.com/

https://apidocs.imgur.com/

How?

如果你没有翻墙软件或代理服务器，可以不用往后看了

使用说明

  1. 注册账号
  2. 新建application以获取接口调用参数 client_id 和client_secret
  3. 安装postman imgur 接口 获取token推荐方式 brew cask install postman
  4. 启动postman
  5. 打开postman后随便点击个api
  6. 填上所需参数
    • https://www.getpostman.com/oauth2/callback
    • https://api.imgur.com/oauth2/authorize
    • https://api.imgur.com/oauth2/token
    • APPLICATION_STATE
  7. 点击 request token

    这里需要注意，postman 配置了proxy 也不会弹出这个页面，需要配置全局代理才起作用

  8. 点击 allow
  9. 得到上述参数后开始配置alfred workflow wntc imgur_use 为true 才能使用 不配或 false 不会上传到imgur
  10. 使用
  11. 快捷键使用 默认快捷键 为 cmd+shift+p ，该方法会将图片上传到所有配置正确的图床， 如果配置了，favor_yun 为 imgur ，则剪贴板默认就是 imgur 的markdown url，如果没配favor_yun，虽然上传了，但是返回的地址第一个上传的云
