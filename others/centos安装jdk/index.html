What

How

  1. 下载jdk http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html wget http://download.oracle.com/otn-pub/java/jdk/8u152-b16/aa0333dd3019491ca4f6ddbe78cdb6d0/jdk-8u152-linux-x64.tar.gz?AuthParam=1513491653_780a1246d58a066130a62b431b8be265

  2. 创建目录 mkdir -p /usr/local/java

  3. 将安装解压到该目录 tar -zxvf jdk-8u152-linux-x64.tar.gz -C /usr/local/java

  4. 配置环境变量 vi /etc/profile

  5. 使配置立即生效 source /etc/profile

  6. 验证是否配置正确 java -version
