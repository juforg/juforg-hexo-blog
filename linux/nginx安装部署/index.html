nginx/tengine编译安装

发表于 2016-09-03 | 更新于 2020-02-05 | 分类于 linux | 评论数： | 阅读次数：

应用场景

http://blog.csdn.net/chenyulancn/article/details/70843696 <!-- more -->

步骤

  1. 切换yum源，使自动寻找最近的可用yum源码

  2. 安装必须依赖

  3. 创建用户和用户组，为了方便nginx运行而不影响linux安全

  4. 下载nginx

  5. 解压nginx

  6. 编译前配置nginx

    • 最好用配置

  7. 开始编译 make && make install

  8. mkdir -p /var/tmp/nginx/client

  9. cd /etc/init.d

  10. vi nginxd

  11. cd /opt

  12. rm -rf tengine-2.1.2

  13. chmod +x nginxd

  14. chkconfig --add nginxd 加入系统服务

  15. chkconfig --list nginxd 查看是否加入成功

  16. 启动,停止,重新装载 service nginxd start|stop|reload /opt/soft/tengine-2.1.2/sbin/nginx -c /opt/soft/tengine-2.1.2/conf/nginx.conf

  17. curl localhost 出现 welcome 就说明 nginx 启动成功了 chkconfig --list nginx

文档

http://tengine.taobao.org/nginx_docs/cn/docs/http nginx.org/en/docs/http

End
